CSS Eğitimi

Css ile web sayfaların tasarımlarını düzenleriz.

Seçiciler

Css ile sayfalarımızdaki elementleri belirli değerlere göre seçip kontrol edebiliriz.

En temel Seçiciler

  1. İd seçiciler : Sayfadaki elementleri id değerine göre seçip css uygulayabiliriz
  2. Class seçiciler : Sayfadaki elementleri class değerine göre seçip css uygulayabiliriz
  3. Element Seçici : Sayfadaki elementleri tag name değerine(div, p, h1-h6, table..) göre seçip css uygulayabiliriz

Css nasıl eklenir

Css kullanmanın 3 yolu vadır
  1. harici css
  2. dahili css
  3. satır içi (inline) css
